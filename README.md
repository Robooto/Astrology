## Webscraping project

#### Backend

The backend will be an express app that scrapes an astrology website and parses that info into json for saving into a db or serving up.

* cd backend
* npm install
* npm start
* npm test

## Routes
- /scopes
    - Returns all horoscope data
- /scopes/:sign
    - Returns the horscope for the sign
